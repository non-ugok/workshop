繰り返し（While文）

繰り返しとは「同じ処理を何度も繰り返すこと」です。
２つある繰り返し構文のうち、まずは「 while文（whileループともいいます）」を紹介します。

Pythonで書くwhile文の構文は次の通りです。
while 繰り返し条件 :
　繰り返したい処理（複数行OK）

それに対して、JavaScriptは…
while ( 繰り返し条件 ) {
　繰り返したい処理（複数行OK）
}
となります。

Pythonでは、繰り返し条件の後にコロン「 : 」を付けるのが特徴ですが、JavaScriptでは、繰り返し条件を「 () 」で囲み、「 {} 」の中に繰り返し処理を記述します。

x = 50
cnt = 1　// 10回数えるための変数

while cnt <= 10 :
　circle(x, height /2, 100)
　x += 100
　cnt += 1 　// 変数cntを1ずつ増やす
let x = 50 ;
let cnt = 1 ; 　// 10回数えるための変数

while (cnt <= 10) {
　circle(x, height / 2, 100) ;
　x += 100 ;
　cnt += 1 ; 　// 変数cntを1ずつ増やす
}
