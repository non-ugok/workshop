テンプレートの使い方

今現在、６種類のテンプレートを用意しています！（今後増やしていく予定です…！）
① circle_free ({});
② circle_paint ({});
③ drawSpiralSquares ({});
④ circle_rotate ({});
⑤ drawShapeGrid ({});
⑥ drawRandomSquares ({});

使い方は簡単で、各オブジェクト名を「Script.jsファイル」で circle_free({}); のように呼び出すだけで利用できます。
また、オブジェクトの引数部分（ {}の中）に size: 80 や speed: 0.05 のように入力することで、図形の大きさや形、動きなどを自由に変更してオリジナル作品を作成できます！

【 例えばこんな感じ… 】
circle_free ({
　r : 150,
　num : 7,
　vx : random(-2, 2),
　vy : random(-2, 2),
　bl : 80,
});

円が自由自在に動く（ circle_free({}); ）

① r : ( 数値 ) → 半径の大きさ
② num : ( 数値 ) → 円の数
③ vx : random( 数値, 数値 ) → X方向のランダム移動範囲
④ vy : random( 数値, 数値 ) → Y方向のランダム移動範囲
⑤ bl : ( 数値 ) → 角丸の大きさ

四角が自由自在に動く（ circle_paint({}); ）

① r : ( 数値 ) → 半径の大きさ
② num : ( 数値 ) → 円の数
③ vx : random( 数値, 数値 ) → X方向のランダム移動範囲
④ vy : random( 数値, 数値 ) → Y方向のランダム移動範囲
⑤ bl : ( 数値 ) → 角丸の大きさ
⑥ baseColors: [color(R, G, B, 透明度), … ] → 図形の色（複数色指定OK）

広がったり、狭まったり（ drawSpiralSquares({}); ）

① size : ( 数値 ) → 図形の大きさ
② num : ( 数値 ) → 円の数
③ speed :( 数値 ) → 図形の動きスピード
④ bl : ( 数値 ) → 角丸の大きさ
⑤ colors: [ color(R, G, B, 透明度), … ] → 図形の色（複数色指定OK）

ぐるぐる回転（ circle_rotate({}); ）

① spd : ( 数値 ) → 図形の回転スピード
② r : ( 数値 ) → 半径の大きさ
③ bl : ( 数値 ) → 角丸の大きさ
④ radiusDec : ( 数値 ) → 半径の収縮量
⑤ minRad : ( 数値 ) → 半径の最小値
⑥ maxRad : ( 数値 ) → 半径の最大値
⑦ col : ( 数値 ) → 色の変更間隔をフレーム数で指定

グリッド（ drawShapeGrid({}); ）

① cols : ( 数値 ) → グリッドの列数
② rows : ( 数値 ) → グリッドの行数
③ shapesize : ( 数値 ) →各形状のサイズ（幅と高さ）
④ cornerRadius : ( 数値 ) →形状の形の丸み
⑤ baseColors : [color(R, G, B, 透明度), … ] → 図形の色（複数色指定OK）
⑥ colorChangeSpeed : ( 数値 ) →色の変化速度
⑦ useStroke : true(またはfalse) →輪郭線の表示設定
⑧ randomizeColor : true(またはfalse) →色をランダムに変化させるかどうか

どんどん増えていく図形（ drawRandomSquares({}); ）

① baseColors : [ color(R, G, B), … ]; → 図形の色（複数色指定OK）
② minSize : ( 数値 ) →図形の最小サイズ
③ maxSize = ( 数値 ) →図形の最大サイズ
④ speed = ( 数値 ) →図形の出てくるスピード
⑤ bl : ( 数値 ) → 角丸の大きさ
